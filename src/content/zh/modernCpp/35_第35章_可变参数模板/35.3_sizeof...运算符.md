# 35.3 sizeof...运算符

我们知道sizeof运算符能获取某个对象类型的字节大小。不过当sizeof之后紧跟...时其含义就完全不同了。sizeof...是专门针对形参包引入的新运算符，目的是获取形参包中形参的个数，返回的类型是std::size_t，例如：

```cpp
include<iostream>   
template<class ...Args>void foo(Args ...args)   
{ std::cout << "foo sizeof...(args) = " << sizeof...(args) << std::endl;   
}   
template<class ...Args>   
class bar   
{ public: bar() { std::cout << "bar sizeof...(Args) = " << sizeof...(Args) << std::endl; }   
}；   
int main() { foo(); foo(1,2,3); bar<> b1; bar<int, double> b2;   
}
```

```txt
foo sizeof...(args) = 0  
foo sizeof...(args) = 3  
bar sizeof...(Args) = 0  
bar sizeof...(Args) = 2
```

# 现代C++学习手册

一个全面的现代C++学习平台,涵盖C++11到C++23的所有重要特性、最佳实践和实战项目。

## 特性

- 📚 **系统化学习路径** - 从C++11到C++23的完整知识体系
- 🔧 **STL深度解析** - 标准模板库的原理与实践
- 🧠 **智能指针与内存管理** - RAII原则和现代内存管理技术
- 🎯 **模板元编程** - 高级模板技术和编译期计算
- ⚡ **并发编程** - 多线程、异步编程和性能优化
- 🤖 **AI辅助学习** - 7x24小时智能答疑和代码解析
- 💻 **丰富代码示例** - 大量可运行的示例代码
- 🚀 **实战项目** - 真实项目案例和工程实践

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/yourusername/HackCpp.git

# 安装依赖
cd HackCpp
pnpm install

# 启动开发服务器
pnpm dev
```
